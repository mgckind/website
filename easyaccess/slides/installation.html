Installation

Making things even easier. and supported, is work in progress.

  • From source, clone repo python setup.py install
  • Using pip or easy_install pip install easyaccess==1.4.6
  • Install with conda conda install easyaccess==1.4.6 -c mgckind
  • Pre-installed in a docker image docker run -it mgckind/easyaccess easyaccess


Press to see installation details.
## Requirements - Oracle Client > 11g.2 (External client library, not python) Check [here](https://www.oracle.com/technetwork/database/database-technologies/instant-client/overview/index.html) for instructions on how to install this client. - [cx_Oracle](https://cx-oracle.readthedocs.io/en/latest/index.html) Note that cx_Oracle needs libaio on some Linux systems Note that cx_Oracle needs libbz2 on some Linux systems - [fitsio](https://github.com/esheldon/fitsio) >= 0.9.6 - [pandas](http://pandas.pydata.org/) >= 0.14 - [numpy](https://docs.scipy.org/doc/numpy-1.15.1/reference/index.html) - [termcolor](https://pypi.python.org/pypi/termcolor) - [PyTables](http://pytables.github.io/) (optional, for hdf5 output) - [future](http://python-future.org/) (for python 2/3 compatibility) - [requests](http://docs.python-requests.org/en/master/) - [gnureadline](https://github.com/ludwigschwardt/python-gnureadline) (optional, for better console behavior in OS X) - **Note that you need to install python-future for python2/3 compatibility**

Source and pip installation

To clone and install from repository https://github.com/mgckind/easyaccess

git clone https://github.com/mgckind/easyaccess.git
cd easyaccess
python setup.py install

instead of the last line:

pip install -e .

Installing using pip: 

pip install easyaccess

Conda installation

For and there is a conda package for easyaccess.

Check here for instructions on how to install conda.

easyaccess conda repository is here. To install it :

conda install -c mgckind easyccess

To create an isolated conda enviroment for easyaccess:

conda create --name ea --yes easyaccess -c mgckind
source activate ea
easyaccess

Using Docker

Sometimes installation of the Oracle client can be tricky. Docker containers provide a lightweight framework to have an isolated environment with pre-installed software. easyaccess repository is here
To run easyaccess:
docker run -it mgckind/easyaccess easyaccess
To persist data in current working directory:
docker run -it -v $PWD:/home/jovyan/work mgckind/easyaccess easyaccess
