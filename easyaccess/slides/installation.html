Installation

Making things even easier. and supported, is work in progress.

  • From source, clone repo python setup.py install
  • Using pip or easy_install pip install easyaccess
  • Using EUPS @ DESDM eups distrib install easyaccess 1.4.0+0
  • Pre-installed in a Docker Image docker run -it mgckind/easyaccess easyaccess
  • New! Install with conda conda install easyaccess -c mgckind


Press to see installation details.
## Requirements - Oracle Client > 11g.2 (External client library, not python) Check [here](https://opensource.ncsa.illinois.edu/confluence/display/DESDM/Instructions+for+installing+Oracle+client+and+easyaccess+without+EUPS) for instructions on how to install this client. - [cx_Oracle](https://bitbucket.org/anthony_tuininga/cx_oracle) Note that cx_Oracle needs libaio on some Linux systems Note that cx_Oracle needs libbz2 on some Linux systems - [fitsio](https://github.com/esheldon/fitsio) >= 0.9.6 - [pandas](http://pandas.pydata.org/) >= 0.14 - [termcolor](https://pypi.python.org/pypi/termcolor) - [PyTables](http://pytables.github.io/) (optional, for hdf5 output) - [future](http://python-future.org/) (for python 2/3 compatibility) - [requests](http://docs.python-requests.org/en/master/) - [gnureadline](https://github.com/ludwigschwardt/python-gnureadline) (optional, for better console behavior in OS X) - importlib (This is only needed if running python 2.6) - **Note that you need to install python-future for python2/3 compatibility**

Source and pip installation

To clone and install from repository https://github.com/mgckind/easyaccess


git clone https://github.com/mgckind/easyaccess.git
cd easyaccess
python setup.py install

instead of the last line:


pip install -e .

Installing using pip: 


pip install easyaccess

Conda installation

For and there is a conda installation that includes Oracle client. Check here to install conda. easyaccess conda repository is here. To install it:


conda install -c mgckind easyccess

To create an isolated conda enviroment for easyaccess:


conda create --name ea --yes easyaccess -c mgckind
source activate ea
easyaccess
