Some cool features

  • Logs a history of commands
  • Smart TAB autocompletion of commands, columns, users, tables, and paths, and more...
  • Very flexible configuration which is available directly through the editor
  • Completely open source and written in Python
  • Write tables to CSV, TAB, FITS, and HDF5 file formats
  • Load SQL queries directly from a file
  • One step to load tables directly into the DB with no memory issues
  • Can be imported as Python module (py2/py3)
  • Tables are handled and displayed using pandas
  • Python functions can be run inside a query (beta)
